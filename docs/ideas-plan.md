# Ideas and Plan — Scenario Reasoner LM

## Vision
Build a training framework for open-source language models to reason into
structured scenarios using chain-of-thought and tree-of-thought prompting
strategies, augmented by real-time monitoring of reasoning quality.

## Current Scope (Scaffold)
- Base infrastructure: datasets, losses, metrics, monitoring, logging
- No concrete model architectures (yet)
- No scenario-type-specific logic (yet)

## Planned Scenario Types (Future)
- Causal / counterfactual reasoning
- Multi-step arithmetic and algebra
- Legal and contract analysis
- Medical decision support
- Game-theoretic scenarios
- Code debugging

## Training Paradigms to Explore
- Supervised Fine-Tuning (SFT) on reasoning traces
- Reinforcement Learning from Human Feedback (RLHF)
- Reward-based CoT / ToT selection
- Self-play and synthetic reasoning trace generation

## Monitoring Extensions
- Integrate monitors into reward signal (penalise/reward CoT quality)
- Track aha-moment frequency as a proxy for "learning velocity"
- Dashboard for real-time CoT/ToT/Aha curves

## Dataset Sources
- Open reasoning benchmarks (ARC, HellaSwag, WinoGrande, etc.)
- Custom scenario corpora
- Synthetic data generated by stronger teacher models

## Evaluation Roadmap
- Exact match, F1 for final answers
- CoT faithfulness metrics
- Scenario-specific rubrics
